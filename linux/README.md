## Virtualization
***KVM*** - linux-based virtualization module. Provide resources (CPU, RAM, GPU, scheduler, etc.) for guest.
- *Guests* - is a virtual OS, worked on a virtualization module. 
- [Habr](https://habr.com/en/post/466549/)
